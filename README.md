# PostHog C
This is an experimental [PostHog](https://posthog.com) C library.

> [!IMPORTANT]
> This is not an official library but only a hobby project.

## Usage
Posthog C uses the [libcurl](https://curl.se/libcurl/) library and [CMake](https://cmake.org/).

```
make
make test
```
